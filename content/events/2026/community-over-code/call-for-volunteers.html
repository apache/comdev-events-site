Call For Volunteers, Community over Code 2026

Community over Code 2026 will be held in Glasgow, UK, October 11-14.

Community over Code is an event run by the community, for the community. That’s you!

We need your help to make it a success!

We are looking for volunteers for a number of community engagement tasks. These roles include:

  • Hackathon Coordination - Help organize and run the hackathon
  • University Outreach - Connect with colleges and universities across multiple departments
  • Meetup Outreach - Engage with local meetup groups
  • Speaker Support - Provide coaching and feedback to speakers
  • Design - Create visual materials for outreach and on-site

If you are interested in volunteering for any of these jobs, please:

  • Join the #volunteers channel on apachecon.slack.com
  • Email planners@apachecon.com indicating your interest, and what you’d like to work on.

There are no specific requirements or qualifications for these jobs - just a passion for ASF community and a willingness to collaborate. We’re looking forward to working with you!
