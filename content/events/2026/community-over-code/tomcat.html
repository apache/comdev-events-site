  • Name — markt@apache.org

What We’re Working On

We will be working on new features and enhancements to Apache Tomcat. The ideas list is being maintained on the project Wiki. We welcome additional suggestions on the dev@ list if you have an idea for something you’d like to work on.

We’re focusing on:

  • Code Quality - Review, and fix where appropriate, issues identified by static analysis such as Coverity scan and GitHub code scanning.
  • HTTP/2 — Better specification compliance for various edge cases and error conditions
  • TLS Documentation - Writing guides for various TLS usage scenarios (Bouncy Castle, FFM)
