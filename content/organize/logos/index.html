Apache Event Logos

The use of Apache Logos and Marks is controlled. For information on using Apache Logos and Marks in general, please see the Apache Trademark Policy. For information on using Apache Logos and Marks for external events, please see the Third Party Event Branding Policy.

For events that have been approved to use Apache Logos (typically events being organized by Apache committers and projects, which have been approved by ConCom), a number of Event Logos are available. Some of these can be used as-is by approved events, others provide a useful starting point when designing logos, especially for print.
