<img src="/images/apache-oak-leaf.svg" alt="Apache" style="height:1.4em; vertical-align:middle;"> **[Community over Code 2026](https://communityovercode.org) — Glasgow, UK, October 11–14**

Community over Code is the annual conference of the Apache Software
Foundation — bringing together developers, community managers, and
enthusiasts from across the ASF ecosystem.

## Get Involved

* **[Hackathon](hackathon.html)** — Dedicated time and space for contributors to
  work together on real project tasks. 17 projects are already signed up.
* **[Call for Volunteers](call-for-volunteers.html)** — Help make the event a
  success — hackathon coordination, university outreach, meetup
  engagement, speaker support, and design.
